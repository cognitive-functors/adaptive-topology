# Code of Conduct

## Contributor Covenant v2.1

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its Code of Conduct.

All participants in this project -- including contributors, maintainers, and users interacting in project spaces -- are expected to uphold a respectful, inclusive, and professional environment.

## Summary

We are committed to providing a welcoming and inspiring community for all. We expect participants to:

- Be respectful and considerate in speech and actions
- Accept constructive criticism gracefully
- Focus on what is best for the community
- Show empathy towards other community members

## Scope

This Code of Conduct applies within all project spaces (issues, pull requests, discussions, code review) and in public spaces when an individual is representing the project.

## Enforcement

Instances of unacceptable behavior may be reported to the project maintainers at:

**Email:** comonoid@yandex.ru

All complaints will be reviewed and investigated. Maintainers are obligated to maintain confidentiality with regard to the reporter of an incident.
