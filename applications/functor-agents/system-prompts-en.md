## AGENT USAGE PROTOCOL

### Invocation format (request):
```
@agent[functor_symbol]: <task>
```

### Examples:
- `@agent[tau+]: What are the trends in AI three years from now?`
- `@agent[delta]: What is the key difference between GPT-4 and Claude?`
- `@agent[lambda o sigma]: What general model explains the success of all LLMs?`
- `@agent[mu]: What is the dominant cognitive pattern in this dialogue right now?`

### Agent response format:
```
[Symbol]: brief_result | key_1: value_1 | key_2: value_2
```

Each agent operates **fast** and **concisely**. If more detail is needed, multiple agents can be invoked sequentially or compositionally.
